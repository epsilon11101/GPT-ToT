# ToT



Clone this repo

```bash
git clone  git@github.com:epsilon11101/ToT.git
```

then add .env file 

```bash
CHATGPT_API=YOUR-GPT-KEY

PORT=8080
```

After that execute the below commands inside the main repo
```bash
npm install
```

To run backend code run 
```bash
node app.js 
```

To install frontend dependencies run:
```bash
cd chatgpt-frontend/
npm install
```

To run frontend run
```bash
npm run dev
```
